# Security Policy
*This security policy is still getting worked on*

## Supported Versions

We take security very seriously. 
All versions are 90%+ secure. 
We do not guaranteed all versions are secure.

| Version | Supported          |
| ------- | ------------------ |
| Latest Release*   | :white_check_mark: |
| Previous Releases   | :x:                |

1 - The latest version will be tested, if a issue occurs we fix it. Bugs and Security issues may slip and be in the final release.
## Reporting a Vulnerability

If you find a issue, please open a ticket in https://d.ziggymc.me/bugs or publish a thread in issues.
